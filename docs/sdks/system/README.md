## Ping

Simple endpoint to verify your API connection is working

### Example Usage

```go
package main

import(
	"context"
	"firehydrant"
	"log"
)

func main() {
    ctx := context.Background()
    
    s := firehydrant.New(
        firehydrant.WithSecurity("<YOUR_API_KEY_HERE>"),
    )

    res, err := s.System.Ping(ctx)
    if err != nil {
        log.Fatal(err)
    }
    if res.PongEntity != nil {
        // handle response
    }
}
```
